This is the repository for the webpage of DMOST, the collaboration working on the ALMA Large
Program 2025.1.00324.L. This webpage is maintained by Nicolas Kurtovic, astronomer at the
Max Planck Institute for Astronomy. If you need to get in contact with me, please check out
my contact information in my personal webpage. Google Nicolas Kurtovic astronomy. 

The original template comes from HTML5 UP (Arcana), with further information provided at
end of this README.txt file. Several modifications have been applied to include card and
hoverable interfaces.
